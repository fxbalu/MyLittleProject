    Getting started

    Ubuntu

    1. Install libsdl1.2-dev, libsdl-image1.2-dev, libsdl-mixer1.2-dev and libsdl-ttf2.0-dev on your system.
    sudo apt-get install libsdl1.2-dev libsdl-image1.2-dev libsdl-mixer1.2-dev libsdl-ttf2.0-dev
    1. Clone the repository.
    git clone https://github.com/fxbalu/MyLittleProject
    1. Create a new SDL project with Code::Blocks.
    2. Add the src folder to your Code::Blocks project.
    3. Add SDL, SDLmain, SDL_image, SDL_mixer and SDL_ttf in Project > Build options... > Linker settings > Link libraries
    4. Build and run the project (F9).
